# Api Jwt

A simple jwt service.

## Prerequisites

* [nodejs](https://nodejs.org/en/)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e
```
